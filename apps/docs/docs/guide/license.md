# License

Reviewskits follows an **Open Core** model. Our core application is open-source under the AGPLv3 license, while advanced enterprise features are available under a commercial license.

## The AGPL Reviewskits Core

The Reviewskits core application is licensed under the **GNU Affero General Public License v3 (AGPLv3)**. 

### What does this mean?
- **Free for Personal & Commercial Use**: You can use the core for free.
- **Modifiable**: You can modify the code.
- **Network Disclosure**: If you run a modified version of Reviewskits on a server and provide it as a service, you **must** make your modified source code available to your users under the same AGPLv3 license.

## The Enterprise Edition

The Enterprise Edition includes advanced functionality designed for larger teams and complex requirements (e.g., advanced analytics, custom integrations, team permissions). 

- **Commercial License**: This part of the codebase is not AGPL.
- **Unlocking Features**: The functionality is built into the official Docker images but requires an **Enterprise License Key** to activate.

## Why this model?

We believe in the power of open-source. Charging for enterprise features allows us to:
- **Build a Sustainable Project**: Funding the team to work on Reviewskits full-time.
- **Keep the Core Free**: Ensuring the foundational tools remain accessible to everyone forever.
- **Long-term Support**: Guaranteeing the infrastructure you rely on will be maintained and improved for years to come.
